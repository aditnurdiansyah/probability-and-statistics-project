## 📋 Overview

This repository contains the source code, case study, and documentation for a **one-sample Z-test** hypothesis testing tutorial implemented in R. The tutorial demonstrates how to perform hypothesis testing when the population standard deviation (σ) is known, using customer satisfaction scenario.

## ▶️ How to Run

### Prerequisites

- [R](https://cran.r-project.org/) (version 4.0 or higher)
- [RStudio](https://posit.co/download/rstudio-desktop/)

### Steps

1. **Clone this repository**
   ```bash
   git clone https://github.com/aditnurdiansyah/probability-and-statistics-project.git
   ```

2. **Open RStudio**
   Go to `File` → `New File` → `R Script` (or open the existing `source-code.R` directly).

3. **Open the script**
   ```
   File → Open File → source-code.R
   ```

4. **Run the script**
   Click **Source** to execute the entire script, or select all (`Ctrl+A`) and click **Run**.

5. **Read the output**
   The console will display all computed values including SE, Z-statistic, critical values, P-value, decision, and conclusion.
